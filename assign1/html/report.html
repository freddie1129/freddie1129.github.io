Summary of Tree-Shrew Publishing Example Website Design

1.Objective

According to the requirements from Tree-Shrew Publishing Company, design a demo website to compete the contract. The website should be simple, attractive, easy to navigate and able to be responsive to different devices, such as desktop, table and mobile.

2.Missions

1) Design a company logo, which should represents the value of the company in a simple, easy to remember and impressive way.

2) Design three main webpage to show the company.

  • Welcome page.
  • Home page, including the introduction section about this company, authors section showing the registered authors in the company and their brief introduction.
  • Author page. Each author has a specific webpage to show biography and books they published, so there would be many author pages depending on the author number. The style of all author pages is same and come from same template.

3.Techniques

1) Using standard HTML5 and CCS3 to implement the whole website, without any third-part framework.

2) Implementing the responsive webpage by elegant HTML5 and CCS3 design, without any other third-part Library support.

4.Design

1) Log Design

Fig 1 Tree-Shrew Publishing Logo

This logo depicts a small boat sailing in the sea. The string “Tree-Shrew Publishing” in this logo can be understood as a wind to push the boat. To emphasize the importance of “Tree-Shrew Publishing” company and make a deep first impression to web visitors, the first letter “T” is colored as red.

3) Responsive webpage design

To maximize the flexibility of this web site for diverse devices, including desktop, tablet and mobile browser, all webpage should be designed in a responsive way, as show in Fig 3.

Fig 3 Responsive Webpage Demo

The main technique to implement the responsive webpage is grid-view design. For large size items shown on the screen, like images, percentage size is used in css instead of the fixed size value. The horizontal layout sections would become vertical layout sections when the screen’s width is less than 786px. The ccs code related to responsive webpage is show as below:

* {
    box-sizing: border-box;
}
/*responsive Design*/
.row::after {
    content: "";
    clear: both;
    display: table;
}
[class*="col-"] {
    float: left;
    padding: 5px;
	margin-right:0px;
}
.col-1 {width: 8.33%;}
.col-2 {width: 16.66%;}
.col-3 {width: 25%;}
.col-4 {width: 33.33%;}
.col-5 {width: 41.66%;}
.col-6 {width: 50%;}
.col-7 {width: 58.33%;}
.col-8 {width: 66.66%;}
.col-9 {width: 75%;}
.col-10 {width: 83.33%;}
.col-11 {width: 91.66%;}
.col-12 {width: 100%;}

@media only screen and (max-width: 768px) {
    /* For mobile phones: */
    [class*="col-"] {
        width: 100%;
    }
}

5.Summary

Tree-Shrew Publishing website design is a simple, attractive and responsive website design. All pages are implemented by Standard HTML5 and CSS3 without other third-part framework. It can suit large desktop screen, as well as small mobile screen. It is easy to navigate and operation. Tree-Shrew Publishing company’s value and business are shown well.
